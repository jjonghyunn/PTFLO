# PTFLO

데이터 분석가 박종현의 개인 포트폴리오 웹사이트입니다.

**[Live Demo](https://ptflo-zeta.vercel.app)**

## Tech Stack

- Next.js 16 (App Router) + React 19
- TypeScript 5
- Tailwind CSS 3.4
- Vercel 배포

## Features

- 한/영 bilingual 지원 (네비게이션 언어 토글)
- 다크 테마 기반 반응형 레이아웃
- 프로젝트 아코디언 UI (카테고리 배지, 태그, 하이라이트)
- OG Image 자동 생성 (소셜 공유 최적화)

## Sections

## Getting Started

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인할 수 있습니다.
